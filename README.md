# PROJET MOGPL - Jules MAZLUM & Zahra SIDDIQUE

Ce projet implémente des solutions pour des problèmes de programmation linéaire et de théorie des graphes.

## Structure des dossiers

### `Partie1/`
- **mogplex.py** : solveur général pour les problèmes linéaires.
- **maxmin.py** : solution utilisant maxmin.
- **s1_optimal.py** : solution optimale pour le scénario 1.
- **s2_optimal.py** : solution optimale pour le scénario 2.
- **minmax_regret.py** : solution utilisant minmax regret.
- **generator.py** : génère des paramètres aléatoires pour tester le solveur.
- **main.py** : exécute des tests sur 10 instances générées.

### `Partie2/`
- **solver2.py** : solveur pour les problèmes linéaires.
- **maxOWA.py** : solution utilisant maxOWA.
- **minOWA_regrets.py** : solution utilisant minOWA_regrets.py.
- **generator2.py** : génère des paramètres aléatoires pour tester le solveur.
- **main2.py** : exécute des tests sur 10 instances générées.
- **plot.py** : fonctions pour tracer.


### `Partie3/`
- **sg1_optimal.py** : solution optimale pour le scénario 1 sur le graphe de gauche.
- **sg2_optimal.py** : solution optimale pour le scénario 2 sur le graphe de gauche.
- **sd1_optimal.py** : solution optimale pour le scénario 1 sur le graphe de droite.
- **sd2_optimal.py** : solution optimale pour le scénario 2 sur le graphe de droite.
- **maxmin_g.py** : solution maxmin pour le graphe de gauche.
- **maxmin_d.py** : solution maxmin pour le graphe de droite.
- **minmax_regret_g.py** : solution minmax regret pour le graphe de gauche.
- **minmax_regret_d.py** : solution minmax regret pour le graphe de droite.
-**solver2V2.py**: criteres owa pour graphes
- **minOWA_g.py** 
- **maxOWA_g.py** 
  
#### `Partie3/statistiques/`
- **graph_generator.py** : génère un graphe aléatoire.
- **matrix_generator.py** : génère une matrice de contraintes pour les arcs.
- **generator.py** : génère des paramètres aléatoires pour tester les solutions.
- **main.py** : exécute des tests sur 10 instances générées.
